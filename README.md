# Gestfy

O Gestfy é um sistema integrado de vendas e gestão desenvolvido para pequenas empresas do setor de alimentação, com foco em sorveterias e lanchonetes que precisam de mais organização, controle e autonomia. A plataforma reúne, em um único ambiente, um site próprio de vendas online, controle automatizado de estoque, registro de caixa, organização de pedidos e relatórios financeiros em tempo real.
